# Krist 2.0

This is the official Krist 2.0 node repository. It is written in Node.js.

## Dependencies

- `colors`
- `express`
- `sequelize`

_Note:_ `mysql` is required if you use `mysql` or `mariadb` as a Sequelize backend.
